# NY Session Adaptive Volatility Squeeze & Momentum Burst

**XAUUSD M1, NY 8–17 EST.**

**Dual Keltner:** Inner (20, 1.5×ATR) for squeeze, Outer (40, 2.5×ATR) for context/TP. **Squeeze:** price inside narrow Inner for 3–5 bars (optionally Inner inside Outer). **Breakout:** strong candle closes above Inner upper (long) or below Inner lower (short), body ≥ 1.5× avg squeeze body. **FMC:** Stochastic 5,3,3 cross (%K above %D for long). Entry at close of breakout candle. SL below/above trigger candle; TP = 1.5–2.5 R or Outer band.
