# Dynamic Liquidity Flush Reversal (DLFR)

**XAUUSD 1-min, NY 8–16 EST.**

**SA-VWAP** from NY session start; **DVB** = SA-VWAP ± ATR(20)×mult. Price flushes into upper/lower DVB with **volume anomaly** (vol ≥ 1.5× avg) and **rejection candle** (close in bottom 25% for short, top 25% for long). Enter on **confirmation candle** (next candle continues the reversal). SL above rejection high + 0.5×ATR (short) or below rejection low − 0.5×ATR (long). TP = SA-VWAP (mean reversion). Optional: partial close at 1.5–2 R, trail 0.8×ATR.
