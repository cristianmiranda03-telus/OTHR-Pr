# Volatile Whisper Retest

**XAUUSD 1–2 min, NY session.**

Micro-structure breakout → retest with **“Volatile Whisper”** (current ATR < SMA(ATR,20) at retest, small retest candles) → strong **rejection candle** → entry. Dynamic SL (rejection low/high + ATR buffer), TP ≥ 1.5 R. **Profit protection exit:** ATR spike above baseline or 2–3 consecutive decreasing body candles after 1R profit.

- Consolidation: 5–15 bars, then breakout (body > 70% range).
- Retest candle(s) at breakout level: small body, short wicks; ATR < baseline.
- Rejection: bullish (long) or bearish (short) with clear wick/body.
