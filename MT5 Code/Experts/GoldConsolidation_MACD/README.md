# Gold M5 Consolidation Breakout with MACD Momentum

**Asset:** XAU/USD · **Timeframe:** M5

- **Consolidation:** range over the last N bars (e.g. 5), minimum height 15–20 pips.
- **Breakout:** strong M5 candle closes fully above range high (long) or below range low (short); body outside the range.
- **MACD 12,26,9:** long when histogram is positive and current bar > previous (rising momentum); short when histogram is negative and current bar < previous.
- **Entry:** at the open of the candle following the breakout candle.
- **SL:** 18–20 pips from entry (below entry for long, above for short).
- **TP:** 25–30 pips from entry.

Attach to XAU/USD M5. Best used in high-liquidity sessions. Adjust `InpPipsPerPoint` (often 10) for your broker.
